Musify

A full-stack web application built with Next.js and Nest.js allows you to upload music singles or albums with cover art, and then you can access and listen to them. There is also an option for user registration to add albums to favorites.

TypeScript React NextJS NestJS MongoDB Redux, RTK Query
Kinoteka

Also a full-stack SPA built with React and Express. Movie catalog. If you're logged in, you can add the movie info, and it will be uploaded to my server. There is also a watchlist where you can add movies. Backend API allows you search with query parameters, so you can share a link with your friends.

TypeScript React Express Redux, RTK Query MongoDB JWT React Router
World

Simple frontend application showing you world map. Used REST countries API. Map made with SVG,worked with zoom in and out whick you can use with mouse or trackpad.

TypeScript React SVG Axios
Pomodoro

Pomodoro timer made with pure JS. One important thing: fully accessible. You can use the app without a mouse, only with a keyboard. A settings popup also. A simple design and an alert sound at the end of the timer would help you focus on studying or working.

JavaScript Gulp Accessibility Bem
